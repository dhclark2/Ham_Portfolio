CSS

  1. Change background color
    For the color, we are using CSS3 the nth-child(). For the normal background, we using white color #fff, and we markup with others color using the nth-child(odd) on every section in div. This means, it will change color on odd number. Please look at style.css at line 464. There we start on the opacity: 0 and when the user scroll down, which is the .modify, line 475, we change the opacity to 1 and the color of the odd number of section div. You can change each div color at line 477.

    * Plase note that, we put the background color on the .modify because we donâ€™t want any of the color went out before the transition.
  2. Change section background color
    To change a section background color like we did in count section, we also use the nth-child(), but instead of odd, we put number 5. Which means the div section of count is number 5. Lets count:
